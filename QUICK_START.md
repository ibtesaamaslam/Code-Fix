# 🚀 Code Fix - Quick Start Guide

## ⚡ Super Quick Start (One Command)

```bash
npm run quick-start
```

This will automatically:
1. Install all dependencies
2. Start the development server
3. Open the application at http://localhost:3002

---

## 📋 Manual Setup (Step by Step)

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open Your Browser
Navigate to: **http://localhost:3002**

---

## 🎯 Available Scripts

| Command | Description |
|---------|-------------|
| `npm install` | Install all dependencies |
| `npm run dev` | Start development server with auto-reload |
| `npm start` | Start production server |
| `npm run quick-start` | Auto-install and start (recommended) |
| `npm run build` | Prepare for production |
| `npm run deploy` | Build and deploy |

---

## 🎨 How to Use Code Fix

1. **Write Code**: Enter your code in the textarea
2. **Select Language**: Choose from JavaScript, Python, TypeScript, Java, or SQL
3. **Analyze**: Click "Analyze & Fix Code" button
4. **Review Results**: See issues, fixes, and suggestions
5. **Copy Fixed Code**: Use the copy button to get the improved code

---

## 🔧 Development Features

- **Auto-reload**: Server restarts automatically when files change
- **Hot-reload**: Browser refreshes when frontend files are updated
- **Error handling**: Graceful error messages and recovery
- **Health monitoring**: Built-in health check endpoints
